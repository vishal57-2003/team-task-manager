## Recording Tips:
1. **Screen Resolution**: Use 1920x1080 for best quality
2. **Audio**: Clear microphone, no background noise
3. **Pace**: Speak clearly and not too fast
4. **Preparation**: Test all features before recording
5. **Backup**: Have the local version running in case of deployment issues
6. **Time**: Keep it between 2-5 minutes as required
